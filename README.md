Shopping Journal ir Android lietotne pirkumu pierakstīšanai, kopējās summas aprēķināšanai, datu saglabāšanai datubāzē, datu nosūtīšanai uz API un foto uzņemšanai.

Projekts izstrādāts studiju uzdevuma ietvaros.

Funkcionalitāte:
✔️ Pievienot pirkumu (nosaukums + cena)
✔️ Datu validācija (nedrīkst saglabāt tukšus laukus)
✔️ Automātiska kopējās summas aprēķināšana
✔️ Datu saglabāšana SQLite (Room) datubāzē
✔️ Datu ielāde pēc aplikācijas restartēšanas
✔️ Datu nosūtīšana uz API ar POST pieprasījumu
✔️ API atbildes attēlošana lietotnē
✔️ Foto uzņemšana ar kameru un saglabāšana ierīcē
✔️ Pēdējā uzņemtā foto attēlošana lietotnē
✔️ Darbs ar Android permissions
✔️ Projekts versiju kontrolē ar GitHub

Tehnoloģijas:
Java
Android Studio
SQLite (Room Database)
Volley (HTTP pieprasījumiem)
